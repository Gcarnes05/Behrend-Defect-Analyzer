
# Behrend Defect Analyzer

This repository provides a workflow for constructing and analyzing charged defects in materials using VASP. The goal is to compute defect formation energies and determine the intrinsic Fermi level at which the system becomes charge neutral. A more detailed step-by-step guide is provided in the BDA tutorial.

To fully utilize this repository, you will need to download the script `sxdefectalign`, written by Christopher Freysoldt. This tool is used to calculate correction terms related to the periodic boundary conditions in VASP simulations. It also accounts for the long-range electrostatic potentials.

Another helpful package is `pydefect`, which provides useful tools for determining the chemical potentials of the constituent elements.

 The general format of defect directory names should be as follows:

- added element _ removed element _ charge state.

Here is an example for each defect type:
- Vacancy: Va_Ga_0
- Substitution: N_Ga_1
- Interstitial: Ga_i1_-1 (Gallium interstitial in i1 position, this name can be changed)

## generate_defects.py

### Description
This script will generate vacancy and substitution defects in a given POSCAR. The location of the defect will be written at the top of the POSCAR file, this is needed for the `sxdefectalign ` script. This currently does not account for different bonding; it just picks the most central atoms of each type. This script would mostly be useful for ternary materials.

### Input Files
BULK VASP POSCAR (Supercell with no defects)

## charge_defect_generator.sh
### Description
This script will take neutral charge state defects and generate charge defects from the given files. The script needs to be given the directory name without the charge state. For example, for a directory named Va_Ga_0 the input would be Va_Ga. 

### Input Files
Name of defect and proper directory setup.

## run_sxdefectalign.sh
### Description
This script runs the `sxdefectalign` script in each directory where the script is run. This will format all of the bulk energies into one file, get corrections for periodic cell boundaries, and generate vAtoms files in each directory. This reads the defect location from the neutral charge state POSCAR at the top of the file.

### Input Files
This will need the dielectric tensor of the material, the location of the bulk (non-defected) LOCPOT file, the location of `sxdefectalign` script on terminal/pc, and a properly formatted directory with LOCPOTs for each defect.

## make_vatoms_output.sh
### Description
This script gets the data from all of the vAtoms files generated by `sxdefectalign` and format it into one file. 

### Input Files
Needs vAtom file in each directory.

## energies_final_vAtoms_plots.py
### Description
Using the files above, this script will determine the correction values due to the difference between the assumed charge potential by VASP compared to the actual values. It will plot these potentials for the user's inspection. The script also combines all correction energies into one file called `energies_final.csv`.

### Input Files
This will need the vAtoms_output.csv, energies_correction.csv, and a POSCAR.

The run command for this file would look like:
```
python energies_final_vAtoms_plots.py -mu -2.91250895 -8.31707533 -percent 0.85
```
There are additional tags that can be seen by using:
`python energies_final_vAtoms_plots.py -help`

## formation_vs_fermi.py
### Description
Using the files above, the charge neutrality of the system will be analyzed, and the intrinsic fermi energy will be determined. The defect formation energy will be plotted as a function of the fermi energy. 

### Input Files
This will need the target_vertices.yaml, energies_final.csv, and a POSCAR.

The run command for this file would look like:
```
python formation_vs_fermi.py -mu -2.91250895 -8.31707533 -bg 1.7378 -vbm 3.4099 --save_as name
```
There are additional tags that can be seen by using:
`python formation_vs_fermi.py -help`
